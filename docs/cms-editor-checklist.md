# CMS Editor Checklist

Use this as a quick routine when updating the website.

## Before Editing

- Confirm which page or section needs updating
- Prepare the final text first
- Prepare any images you want to use

## Upload Images

- Open `Media`
- Upload new images first
- Add meaningful alt text

## Update Shared Info

Open `Site Settings` if you need to change:

- phone
- email
- WhatsApp
- address
- office hours
- navigation links
- footer links

## Update The Homepage

Open `Home Page` to edit:

- hero title
- hero subtitle
- hero image
- trust indicators
- stats
- feature cards
- CTA buttons

## Update Standard Pages

Open `Pages` and select the correct slug:

- `about`
- `services`
- `why-choose`
- `contact`
- `courses`
- `study`

Check:

- hero text
- section titles
- paragraphs
- cards
- bullets
- CTA labels and links

## Update Courses

Open `Courses` to:

- add new courses
- remove old courses
- adjust levels
- update credits

## Update Programmes

Open `Programmes` to:

- add or edit programme cards
- update descriptions
- update feature lists

## Before Saving

- Check spelling
- Check punctuation
- Check links
- Check that no list item is empty

## After Saving

- Refresh the page on the website
- Confirm the new content appears
- Test all changed links
- Check images load correctly

## If There Is A Problem

- Re-open the entry and confirm it was saved
- Make sure you edited the correct slug or collection
- Check if a required field was left blank
- Verify images were uploaded before selection
